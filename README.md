
#My dotfiles. 
============

Managed with stow (wonderful symlink farm).

##INSTALL


```bash
git clone https://github.com/liloman/dotfiles.git 
cd dotfiles 
./install.sh -d  (for desktop use)
./install.sh -s  (for server use)
```
